01

How it works

three moves, no model ids
declare intent

Say what it needs

Requirements, not model names — modalities, features, token estimates.

await router.generateObject({
  task: "recipe.extract",
  inputModalities: ["text","image"],
  requiresFeatures: ["structured_output"],
})
set policy

Make the rules

Cost ceilings, latency, benchmarks, or an LLM judge — enforced at plan time.

preference: "cheapest",
constraints: {
  maxCostUsd: 0.01,
  maxLatencyClass: "standard",
}
execute

Fallback is free

It walks the ranked routes until one succeeds, with the attempt log.

// attempts
[{ model: "gemini", ok: true }]
// or → next route on fail
